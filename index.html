// 01 — Mission Profile

Overview

This project implements a deterministic EKG monitoring system on an ESP32 using FreeRTOS, replicating the high-reliability firmware architecture found in clinical electrocardiogram monitors. To ensure patient safety, the system utilizes a preemptive multitasking kernel where critical tasks—such as ADC signal acquisition and tachycardia/bradycardia detection are granted high priority to meet strict real-time deadlines. Concurrency is managed through the use of semaphores and mutexes, preventing race conditions while coordinating the sensor logic, a hardware-interrupt-driven nurse call button, and a background web server. Beyond real-time alerting, the firmware maintains data integrity by processing peak, low, and average BPM metrics over a rolling 5-minute window. This historical log and current vitals are broadcast to a live web interface, demonstrating a complete end-to-end integration of embedded RTOS principles, hardware simulation via Wokwi, and IoT telemetry.

Signal Acquisition

High-priority ADC sampling simulates ECG input, ensuring no data points are dropped during processing.

Mutex Protection

Heart rate data is a shared resource; Mutexes prevent race conditions between the sensor task and the display task.
